#헤드퍼스트 자바 3판  
다시 공부 하는 자바 
- IDE : vscode
- 자바 : openjdk version "17.0.7"  
-----------이슈 기록-------------
1. 한글이 포함되어 있는 경우 컴파일시 에러나면 다음과 같이 컴파일
```java
javac 파일명.java -encoding utf-8 
```
2. 2장 요약
- 오버라이드(override) : 상속 받은 메서드를 재정의 하는 것
- 인스턴스 변수 : 객체가 자신에 대해 아는 것(상태)
- 메서드 : 객체가 자신이 하는 것(행동)
- 클래스 : 객체를 만들기 위한 설계도
- 객체 : 고유의  변수, 고유의 기능을 지정할 수 있다.
